# Contributing to AI Code Review Squad

Thanks for your interest in contributing! This project is a personal portfolio piece, but I'm always open to feedback and improvements.

## 🚀 Quick Start

1. Fork the repo
2. Create a feature branch: `git checkout -b feature/amazing-feature`
3. Make your changes
4. Add tests if applicable
5. Run tests: `npm test` (frontend) and `pytest` (backend)
6. Submit a PR

## 🧪 Development Setup

```bash
# Clone your fork
git clone https://github.com/your-username/ai-code-review-squad.git
cd ai-code-review-squad

# Set up environment
cp .env.example .env
# Add your API keys to .env

# Run setup
./setup.sh
```

## 📝 Code Standards

- **Python:** Follow PEP 8, use type hints
- **TypeScript:** ESLint + Prettier configured
- **Commits:** Use conventional commit format
- **Tests:** Write tests for new features

## 🤝 Types of Contributions

- 🐛 Bug fixes
- ✨ New agent types or detection rules
- 📚 Documentation improvements
- 🎨 UI/UX enhancements
- ⚡ Performance optimizations
